## 🖥 Instalação

Para instalar o projeto em sua máquina local ou servidor, siga os passos abaixo:

```bash
# Clone o repositório
git clone https://github.com/alemaodacapa/agendamento_visitas.git
cd agendamento-visitas
```

### 📁 Estrutura de Arquivos

```
form.html       → Página principal com formulário  
style.css       → Estilos personalizados da página
```

### 🌐 Hospedagem

- Suba os arquivos em qualquer servidor com suporte a HTML/CSS.
- Para uso com PHP/MySQL, adicione lógica de backend personalizada.
